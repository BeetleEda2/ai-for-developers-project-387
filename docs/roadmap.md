# Roadmap

## Features

### 1. Cancel booking (guest self-service)

**Описание:** Гость может отменить своё бронирование после создания.

**Изменения:**
- Backend: `DELETE /bookings/:id`
- Frontend: кнопка отмены на странице подтверждения

### 2. Отображение слотов в таймзоне гостя

**Описание:** Определять таймзону гостя через браузер и показывать время локально.

### 3. Страница «Мои бронирования» для гостя

**Описание:** Просмотр бронирований по email без авторизации.
